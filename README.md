# fun-apps-to-design-and-build

A collection of small projects focused on system design and implementation — each one explores a specific concept or problem.

| Project | Description |
|---|---|
| [task-scheduler-and-job-queue](./task-scheduler-and-job-queue) | Priority-based job scheduler with cron support, retries, and a Redis-backed queue |
| [parking-lot](./parking-lot) | Parking lot system with slot management and vehicle tracking |
| [tic-tac-toe](./tic-tac-toe) | Tic-tac-toe game with a clean object-oriented design |
| [snakes-and-ladders](./snakes-and-ladders) | Snakes and ladders board game simulation |
| [ui/quiz-blitz](./ui/quiz-blitz) | Real-time multiplayer quiz game UI |
